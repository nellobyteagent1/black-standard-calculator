# Black Standard Calculator

A polished black-themed standard calculator web app built with plain HTML, CSS, and JavaScript.

## Features

- Addition, subtraction, multiplication, and division
- Decimal input
- Clear all and backspace controls
- Keyboard support
- Clean dark UI with responsive layout
- Graceful handling for invalid/incomplete expressions

## Run locally

From this directory:

```bash
python3 -m http.server 8000
```

Then open:

<http://localhost:8000>
